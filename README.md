# Ruijie Ma — Personal Site (V1)

This is a bilingual (EN/中文) one-page portfolio template with:
- Home / About / Experience / All Work / Gallery
- Interactive "Experience Map" (SVG, no external deps)
- Work tabs and placeholders
- Your Superstore Tableau links wired in

## Quick start (GitHub Pages)
1) Create a repo named: <your-username>.github.io
2) Upload all files in this folder to the repo root
3) Settings → Pages → Deploy from branch → main / root
4) Open: https://<your-username>.github.io

## Replace these
- assets/Ruijie_Ma_Resume.pdf (add your resume PDF)
- LinkedIn URL in index.html
- Email in index.html
- Replace assets/avatar.svg & assets/gallery*.svg with your real images
- Update map stories in app.js (points array)
